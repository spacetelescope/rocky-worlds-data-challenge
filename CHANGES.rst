Revision History
================

1.0 - Initial Public Release
------------------------------

This is the initial public release of ``rocky-worlds-data-challenge``.

Included in this release:

* Tools for representing eclipsing systems and deriving related orbital
  quantities.
* Helpers for validating posterior samples, photometry arrays, questionnaire
  responses, and complete submission ZIP archives.
* A blank submission questionnaire template packaged as
  ``rocky_worlds_data_challenge/data/form.json``.
* An interactive Solara form interface for completing questionnaire responses.
* Tutorial material for preparing a challenge submission.
* Documentation, tests, and continuous-integration configuration for public
  development.

Simulation-generation code is not included in this release. The ``simulator``
directory documents the intention to provide that code after the data challenge
is complete.
