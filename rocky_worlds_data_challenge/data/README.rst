Package Data
============

This directory contains small data files that are shipped with
``rocky_worlds_data_challenge``.

Files
-----

``form.json``
    Blank submission questionnaire template used by
    ``rocky_worlds_data_challenge.results.Form`` and the interactive Solara
    form widget. The template defines question prompts, descriptions, required
    status, expected response formats, examples, and initially empty response
    fields.

Large simulation products and challenge data files should not be stored here.
Those files should be hosted externally or placed in a purpose-specific
directory with clear release instructions.
